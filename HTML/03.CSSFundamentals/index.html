What is HTML ?

HTML stands for Hyper Text Markup Language.it's a markup language that web developers use the content of a webpage(not a programming language)

HTML Consists of the elements that describe different types of content:paragraph,links,headings browsers understand HTML and render HTML code as website

In HTML. each elemensts is made up of 3 parts:

  1. The Opening Tag
  2. The Closing Tag
  3. The actual elements
