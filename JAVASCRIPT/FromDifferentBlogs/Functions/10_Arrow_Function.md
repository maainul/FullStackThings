### Arrow Functions

ES6 arrow functions provide you with an alternative way to write a shorter syntax compared to the function expression.

### Add two number

```js
let add = function (x, y) {
  return x + y;
};
console.log(add(10, 20)); // 30
```

### Arrow Function

```js
let add = (x, y) => x + y;
console.log(add(10, 20)); // 30
```

### We can define return statement

```js
let add = (x, y) => {
  return x + y;
};
console.log(add(10, 20)); // 30
```
